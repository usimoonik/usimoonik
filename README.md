USI Moonik Token (USI)
USI Moonik Token (USI) is a fixed-supply BEP-20 utility token deployed on BNB Smart Chain (BSC).

Smart Contract Properties

Fixed supply created at deployment with no possibility of increase
No mint function – Permanently disabled
No burn function – No mechanisms to reduce supply
No transaction fees or taxes (0% on buys, sells, or transfers)
No blacklist, pause, or restriction mechanisms
No ownership override or admin control – Ownership effectively renounced at deployment
